\# OpenVPN Web Config Generator



Webbasierter Generator für OpenVPN Server- und Client-Konfigurationen.



\## Features

\- Eingabe von Server-IP \& Port

\- Automatische PKI \& Zertifikate

\- Client .ovpn mit eingebetteten Keys

\- Benutzername + Passwort werden erzeugt

\- Starke Verschlüsselung (AES-256-GCM, SHA512)



\## Start

```bash

docker-compose up -d
